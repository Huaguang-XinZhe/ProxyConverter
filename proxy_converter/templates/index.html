代理 API 服务

获取所有代理

端点: /api/proxies

参数:

  • limit: 限制返回的代理数量，0 表示不限制
  • filter: 过滤配置文件的模式
  • check: 是否检查代理可用性，值为 1 表示检查
  • format: 返回格式，支持 json 和 text，默认为 text

示例: /api/proxies?limit=5

获取单个代理

端点: /api/proxy

参数:

  • random: 是否随机选择，值为 1 表示随机选择
  • format: 返回格式，支持 json 和 text，默认为 text

示例: /api/proxy?random=1
